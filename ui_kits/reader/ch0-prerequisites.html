Part 0 · Prerequisites

What you need to start.

You do not need a PhD. You need three concepts: vectors, dot products, and the idea that a function can produce a probability distribution. Run the four self-checks below — if all four feel comfortable, you're ready for §1.

The four self-checks

✓
Vectors as ordered tuples of numbers
$v = [v_1, v_2, \ldots, v_d]$ with $d$ entries — that's it.
→ exercise 1
✓
The dot product
$a \cdot b = \sum_i a_i b_i$. Returns a scalar. Big = aligned, zero = perpendicular.
→ exercise 2
✓
The exponential function
$e^x$ is positive everywhere, monotone increasing, and grows fast.
→ exercise 3
✓
A probability distribution over $n$ items
$n$ non-negative numbers that sum to 1.
→ exercise 4
Exercise 1 — Vector arithmetic

Add, scale, take a length.

Three operations cover 90% of what you'll do with vectors in this book. Try them.

Componentwise add. Componentwise scale. Length is $\sqrt{\sum_i v_i^2}$ (Pythagoras in $d$ dimensions).

Exercise 3 — Exponentials grow fast

$e^x$ at six values.

Figure 3 · $x \mapsto e^x$

Doubling $x$ doesn't double $e^x$ — it squares it. This explosive growth is what makes softmax sharp: tiny score differences become large probability ratios.
